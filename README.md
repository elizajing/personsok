How to Run
----------
<ul>
<li>git clone this repo</li>
<li>Run `npm i`</li>
<li>Create a directory called `wsdl`</li>
<li>Paste the necessary certificate and key files into the `wsdl` directory</li>
<li>Run `npm run dev`</li>
<li>Node version: v18.15.0</li>
</ul>

Run Unit Tests
--------------
Run `npm run test`
